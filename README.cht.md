# PicoTag  
**網球與羽毛球拍穿線紀錄 QR Code 系統**

> 一套簡單、不綁定特定硬體的 QR Code 穿線紀錄系統，適用於網球與羽毛球拍。

## 公開網頁介面
PicoTag 可直接透過公開網頁介面使用：

https://picotag.cc  

公開介面提供完整的 QR Code 產生與紀錄查詢功能。  
後台管理功能僅於自行架設時提供。

> [!TIP]
> 目前的公開網頁介面為正式、可實際使用的入口，並非示範頁面。  
> PicoTag 正處於最後的線上公開測試階段。  
> 待系統穩定後，將會公開原始碼。

## 專案簡介

PicoTag 是一個用於將球拍穿線紀錄數位化的開源工具。

相較於容易模糊、脫落或遺失的手寫貼紙與紙本備註，  
PicoTag 為每一次穿線產生專屬的 QR Code，讓紀錄能被清楚查看、分享並於日後追蹤。

對穿線師而言，這能提供更一致且專業的呈現方式；  
對球拍使用者而言，則能提升資訊透明度與查詢便利性。

---

## 核心功能

- **QR Code 穿線紀錄**  
  每次穿線都會產生唯一的 QR Code，內容包含張力設定、線材與球拍資訊、備註與日期等。

  **範例紀錄頁面**：https://picotag.cc/?id=EHshhlCh

- **預先填寫穿線需求**  
  球拍使用者可事先填寫穿線需求，降低溝通誤差。

- **快速分享**  
  紀錄可透過 QR Code 或網址分享，格式一致且容易閱讀。

---

## 進階功能（自行架設）

於完整系統架設後，將可使用以下進階功能：
- 後台穿線紀錄管理  
- CSV 資料匯出，供進一步分析使用  

自行架設為選用項目，適合需要完整資料掌控與管理彈性的使用者。

---

## QR Code 列印方式

PicoTag 可搭配常見的列印方式使用：

- **18mm 標籤機**  
  可快速列印 QR Code 標籤，直接貼於球拍框上。

  ![18mm 標籤機 QR Code 範例](assets/img/label_maker.jpg)

- **一般印表機**  
  列印資訊頁面後，手動裁剪並黏貼 QR Code。

  ![一般印表機 QR Code 範例](assets/img/printer.jpg)

不需要任何專用或特殊硬體設備。
